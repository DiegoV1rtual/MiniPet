Esta carpeta contiene los sonidos de **final del juego**.

Cuando Mini‑Diego completa todas las horas de juego sin morir, el juego
reproducirá aleatoriamente uno de los archivos de audio situados aquí.
Puedes colocar archivos en formato `.mp3`, `.wav` u `.ogg`.

**Uso:**

- Coloca clips de audio de celebración en esta carpeta (por ejemplo,
  fanfarrias o música de victoria).
- Al ganar el juego (cuando el temporizador llega a cero), se seleccionará
  aleatoriamente uno de estos clips para reproducirlo antes de mostrar el
  panel de felicitación.
- Si no hay archivos o la biblioteca de audio no está disponible, no se
  reproducirá ningún sonido y el juego continuará con el panel de victoria.
