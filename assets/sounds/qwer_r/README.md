Esta carpeta contiene los sonidos que se reproducen al pulsar la tecla **R** en el minijuego QWER Hero.

Añade aquí archivos de audio en formato `.mp3`, `.wav` o `.ogg`.  El juego seleccionará uno al azar cada vez que pulses la tecla R.

Si la carpeta está vacía o no hay biblioteca de reproducción instalada, no se reproducirá ningún sonido.
