Esta carpeta contiene los sonidos que se reproducen al pulsar la tecla **W** en el minijuego QWER Hero.

Puedes colocar aquí archivos de audio en formato `.mp3`, `.wav` o `.ogg`.  El juego seleccionará uno al azar cada vez que pulses la tecla W.

Si la carpeta está vacía o no hay biblioteca de reproducción instalada, no se reproducirá ningún sonido.
