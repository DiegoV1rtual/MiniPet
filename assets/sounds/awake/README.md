La carpeta ``assets/sounds/awake`` contiene clips de audio que se reproducen de forma aleatoria mientras Mini‑Diego está despierto.

**Uso:**

- Coloca aquí archivos de sonido en formato `.mp3`, `.wav` o `.ogg`.
- El juego seleccionará uno al azar cada 3–5 minutos cuando Mini‑Diego esté despierto.
- Si la carpeta está vacía o no hay biblioteca de reproducción instalada, no se reproducirá ningún sonido.

Esto permite que la mascota "hable" o emita sonidos al azar, dando más vida al juego.
