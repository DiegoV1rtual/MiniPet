Esta carpeta está destinada a contener una imagen de **fondo** para el
panel de victoria del juego.  Cuando completes todas las horas de
Mini‑Diego sin morir, el juego intentará cargar la primera imagen
encontrada en este directorio y utilizarla como fondo de la pantalla de
felicitación.

## Cómo usar

1. Coloca una imagen aquí con cualquiera de las extensiones
   compatibles: `.png`, `.jpg`, `.jpeg` o `.gif`.
2. La imagen se redimensionará automáticamente para ajustarse a la
   ventana de victoria (700×500 px).  Un fondo con proporción similar
   se verá mejor.
3. Si la carpeta está vacía o la imagen no puede cargarse, el juego
   usará un fondo azul oscuro predeterminado.

Puedes cambiar la imagen en cualquier momento; simplemente reemplaza el
archivo por otro y vuelve a ejecutar el juego.
